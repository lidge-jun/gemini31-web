Made for credentialed local use.

Configure Vertex

Set GOOGLE_APPLICATION_CREDENTIALS, optional VERTEX_PROJECT, and VERTEX_MODEL.

Run locally

Install google-auth, run python3 server.py, then open localhost:3131.

Keep history private

Chats are JSON files under chats/ and should not be committed.

Small server, large browser surface.

Python

server.py handles auth, Vertex calls, SSE relay, config, and chat JSON CRUD.

Browser

index.html contains the vanilla JS UI, markdown rendering, multimodal input, and settings panels.

Pages

GitHub Pages hosts docs only. The live UI requires private credentials and should stay local.

Secrets and chats stay off GitHub.

Secrets

Never commit `.env` or the Vertex service account JSON file.

Config writes

/api/config can update `.env`; run the app only on a trusted local network.

Verification

python3 -m py_compile server.py validates the server without requiring credentials.
